Софт для протеуса работает под линуксом,
поэтому придется поставить второй системой линукс.
У меня стоит вот такой: https://neon.kde.org/download
или вот такой: https://kubuntu.org/getkubuntu/


Папка 0.1 содержит первую версию софта, без визуального
интерфейса. 

Папка 0.2 gui содержит код для поста управления с 
визуальным интерфейсом. 


Основные библиотеки которые  необходимо установить:
- pip3 install PyQt5
- pip3 install pyPS4Controller
- pip3 install keyboard
- pip3 install coloredlogs

!!! модуль опроса клавиатуры не поддерживает работу под обычным пользователем, поэтому надо 
запускать софт через sudo !!!

Сейчас трансляция видео потока вынесена в отдельное окно, в идеале 
интегрировать его в основное там в принцепе не должно возникнуть сложностей. 

Так же не весь интерфейс работает, его надо добить ( приветсвуются предложения по модернизации 
и усовершенствованию)

Неплохой доработкой будет реализация темпой и светлой темы. Так же должена быть реализована возможность смены цвета
не выходя из программы.
