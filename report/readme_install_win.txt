    (при установки питона необходимо скачать)
    https://www.python.org/downloads/


2) Установить VScode (опционально):
    https://code.visualstudio.com/download

    2.1) установка расширений для работы с python

3) Установить гит:
    https://git-scm.com/download/win

    3.1) настроить параметры username и email 
        git config --global user.name имя 
        git config --global user.email почта 
    
    3.2) склонировать репозиторий с прошивкой для Proteus 

3) Установить необходимые библиотеки для работы прошивки:
    (выполнить в командной строке)
    pip install coloredlogs
    pip install pygame
